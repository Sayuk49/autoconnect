#How To Use
 *Kali Linux
1.git clone https://github.com/Sayuk49/autoconnect
2.cd autoconnect
3.bash connect.sh
4.Insert requirements that needed(ex:IP,path)
5.Open new terminal
6.cd autoconnect
7.bash start.sh
8.Insert IP again & Done!. Your serveo is ready.
